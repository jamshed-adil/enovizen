# Enovizen – Integrated Facility Management Services

A React + Tailwind CSS website for Enovizen.

## 🚀 Quick Start

### Prerequisites
- [Node.js](https://nodejs.org/) v18+ installed

### Steps

1. **Unzip** the downloaded file and open the `enovizen` folder in your terminal.

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser at **http://localhost:5173**

---

### Build for production
```bash
npm run build
```

### Preview production build
```bash
npm run preview
```
